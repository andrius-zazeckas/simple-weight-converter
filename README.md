# Simple Weight Converter

This repository contains a basic web application for converting weights between different units. The project uses HTML, CSS, and JavaScript to create a user-friendly interface for performing weight conversions.

## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/andrius-zazeckas/simple-weight-converter.git
2. Navigate to the project directory:
   ```sh
   cd simple-weight-converter
3. Open index.html in your favorite web browser.

## Project structure

- index.html: The main HTML file for the web application.
- script.js: The JavaScript file containing the conversion logic.
- style.css: The CSS file for styling the web application.
